
Android Login App
Description:
This is an Android Login Application built in Java using Android Studio.
The app allows users to log in using hard-coded credentials and provides additional features like registration, password reset, and a landing/home page after login.
Features:
Login Screen:
Authenticate with hard-coded username and password
Shows error message if credentials are invalid
Registration Page:
Users can create a new account (for demonstration only, no database)
Forgot Password Page:
Users can retrieve/reset password
Reset Password Page:
Update password functionality
Landing/Home Page:
Displayed after successful login
Simple welcome message or dashboard
Navigation:
Intents used for screen transitions
“Register” and “Forgot Password” links/buttons on login screen
